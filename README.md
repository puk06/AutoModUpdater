# AutoModUpdater
AutoModupdater for Hypixel Skyblock

# 使い方 
Node.js持ってないならだうんろーどしてね。
ダウンロードが終わったら、コマンドプロンプトを開いて、これを入力してね
```
cd (Modupdater.jsファイルが有るパス)
```
```
npm i
```
クライアントID、クライアントシークレットを使う場合は
```
npm i dotenv -g
```
これも追加してください。
あとは"アップデートしたいMOD"フォルダにアップデートしたいModを入れて終わりです。
